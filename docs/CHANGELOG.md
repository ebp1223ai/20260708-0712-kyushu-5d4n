# Changelog

All notable changes to this project should be documented here.

## v1.0

Initial production version.

Completed:

- GitHub repository setup.
- Vercel production deployment.
- Next.js 15 App Router application.
- TypeScript and Tailwind CSS setup.
- Mobile-first trip website.
- React Leaflet map with OpenStreetMap tiles.
- Location markers from `data/locations.json`.
- Route timeline from `data/routes.json`.
- Day summaries from `data/days.json`.
- Google Maps place and direction links.
- Production URL verified on desktop and mobile.
