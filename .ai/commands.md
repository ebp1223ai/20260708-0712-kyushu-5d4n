# AI Commands

## Feature

Use `prompts/feature.md` or `.ai/templates/feature.md`.

Checklist:

- Read context.
- Keep scope to one feature.
- Build.
- Commit.
- Push only when requested.

## Bugfix

Use `prompts/bugfix.md` or `.ai/templates/bugfix.md`.

- Keep fix narrow.
- Build.
- Commit.
- Push only when requested.
