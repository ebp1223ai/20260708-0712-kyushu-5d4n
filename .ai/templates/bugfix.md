# Bugfix Template

## Checklist

- [ ] Read `docs/PROJECT.md`.
- [ ] Read `docs/AI_CONTEXT.md`.
- [ ] Read `.ai/context.md`.
- [ ] Identify the bug.
- [ ] Keep the fix narrow.
- [ ] Preserve existing behavior outside the bug.
- [ ] Do not change `data/*.json` schema unless requested.

## Build

```bash
npm run build
```

## Commit

Use one focused bugfix commit.

## Push

Push only when requested.
