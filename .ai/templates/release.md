# Release Template

## Checklist

- [ ] Read `docs/RELEASE.md`.
- [ ] Read `docs/DEPLOYMENT.md`.
- [ ] Read `docs/CHANGELOG.md`.
- [ ] Confirm production branch.
- [ ] Confirm production URL.
- [ ] Run build.
- [ ] Update release notes if needed.

## Build

```bash
npm run build
```

## Commit

Use one release-focused commit.

## Push

Push to `origin/main` only after build succeeds.
