# Feature Template

## Checklist

- [ ] Read `docs/PROJECT.md`.
- [ ] Read `docs/AI_CONTEXT.md`.
- [ ] Read `.ai/context.md`.
- [ ] Confirm the feature scope.
- [ ] Inspect relevant files.
- [ ] Implement only this feature.
- [ ] Do not change `data/*.json` schema unless requested.
- [ ] Do not add npm packages unless approved.

## Build

```bash
npm run build
```

## Commit

Use one focused feature commit.

## Push

Push only when requested.
