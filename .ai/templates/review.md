# Review Template

## Checklist

- [ ] Read `docs/PROJECT.md`.
- [ ] Read `docs/AI_CONTEXT.md`.
- [ ] Read `.ai/context.md`.
- [ ] Inspect the requested diff or files.
- [ ] Prioritize bugs, regressions, missing tests, and deployment risks.
- [ ] Include file and line references.

## Output

Lead with findings, ordered by severity.

Then include:

- Open questions.
- Test gaps.
- Short summary if useful.

## Build

Run build only when review scope requires verification:

```bash
npm run build
```
